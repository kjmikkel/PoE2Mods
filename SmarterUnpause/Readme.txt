Smarter Unpause - 2.0.0

:: REQUIREMENTS ::
* Deadfire at version 5.0.0.0040
* Unity Mod Manager: https://www.nexusmods.com/site/mods/21

:: FEATURES ::
This mod makes the auto-pause smarter by having a grace period between the auto-pause happening, and accepting a manual unpause command. By default this value is 0,5 seconds, but you can set it to anything you like in the included config.ini file.

::  CREDITS ::
This is a heavily modified version of the PoE1 mod fo the same name by veyn (which can be found here https://www.nexusmods.com/pillarsofeternity/mods/185).

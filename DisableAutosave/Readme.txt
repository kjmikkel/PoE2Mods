Disable Autosave - 1.0.0.5

:: REQUIREMENTS ::
• GOG version at 5.0.0.0040 and Steam at 5.0.0.0040
• Unity Mod Manager: https://www.nexusmods.com/site/mods/21

:: FEATURES ::
This mod disables all the autosave functionality.

::  CREDITS ::
This is a lightly modified version of a mod made by SonicZentropy (and its original source can be found here: https://github.com/SonicZentropy/PoE2Mods.pw).

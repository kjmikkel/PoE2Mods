Ship Morale - 2.0.0

:: REQUIREMENTS ::
* Deadfire at version 5.0.0.0040
* Unity Mod Manager: https://www.nexusmods.com/site/mods/21

:: FEATURES ::
This allows you to to set the upper and lower floor of the ship crew morale, as well as just giving yourself a certain amount of it

::  CREDITS ::
This is a heavily modified version of a mod made by SonicZentropy (and its original source can be found here: https://github.com/SonicZentropy/PoE2Mods.pw).
